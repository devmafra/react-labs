# desafio-banco-dados

## Descrição

Este projeto React simula a busca assíncrona de dados de um "banco de dados" local (`coursesDB.js`) usando os hooks `useState` e `useEffect`, exibindo um estado de carregamento e, posteriormente, os dados em uma tabela.

## Tecnologias

- React
- Vite
- ESLint

## Como rodar

```bash
npm install && npm run dev
```
